# Titanium Intercom SDK

Support for the native iOS / Android Intercom SDK in Appcelerator Titanium.

## Requirements

- [x] Titanium SDK 9+
- [x] iOS 10+ and Android 5+
- [x] A valid iOS/Android API key and app ID
